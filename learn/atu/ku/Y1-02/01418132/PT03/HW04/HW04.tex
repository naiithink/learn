	\item จงหาจำนวนสมาชิกของ $|A \cup B|$ เมื่อ $|A| = 12$ และ $|B| = 18$ เมื่อแต่ละกรณีต่อไปนี้เป็นจริง

		\item $|A \cap B| = \phi$
		\item $|A \cap B| = 1$
		\item $|A \cap B| = 6$
		\item $A \subseteq B$
	\end{enumerate}
